## Introduction

This is my collection of projects to satisfy the requirements for the [x86 Assembly Language and Shellcoding on Linux (SLAE)](https://www.pentesteracademy.com/course?id=3) course.

The majority of my posts are in markdown format and can be read straight out of [my SLAE x86 github repository](https://github.com/alecmaly/slae/tree/master/slae86), which also hosts the source code for each assignment.



# Assignments

## [Assignment 1](https://github.com/alecmaly/slae/tree/master/slae86/project1)

- Create a Shell_Bind_TCP shellcode
- Binds to a port
- Execs Shell on incoming connection
- Port number should be easily configurable

## [Assignment 2](https://github.com/alecmaly/slae/tree/master/slae86/project2)

- Create a Shell_Reverse_TCP shellcode
- Reverse connects to configured IP and Port
- Execs shell on successful connection
- IP and Port should be easily configurable

## [Assignment 3](https://github.com/alecmaly/slae/tree/master/slae86/project3)

- Study about the Egg Hunter shellcode
- Create a working demo of the Egghunter
- Should be configurable for different payloads

## [Assignment 4](https://github.com/alecmaly/slae/tree/master/slae86/project4)

- Create a custom encoding scheme like the “Insertion Encoder” we showed you
- PoC with using execve-stack as the shellcode to encode with your schema and execute
